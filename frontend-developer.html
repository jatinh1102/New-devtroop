Job Description :

  • Join us as a Frontend Developer to create captivating user experiences. Utilize your expertise in HTML, CSS, and JavaScript to craft seamless interfaces and shape our web presence. If you're passionate about design and collaboration, apply now to make an impact with our dynamic team!
  • As a frontend developer, you'll collaborate closely with frontend developers to seamlessly integrate their work with our Node.js application.

Skills :

  • Proficiency in HTML5
  • Strong understanding of CSS3 and preprocessors like SASS or LESS
  • Advanced knowledge of JavaScript, including ES6+ features
  • Experience with frontend frameworks/libraries such as React, Vue.js, or Angular
  • Familiarity with responsive design principles and techniques
  • Ability to work with RESTful APIs and AJAX
  • Version control systems like Git
  • Understanding of browser compatibility issues and solutions
  • Knowledge of build tools like Webpack or Gulp
  • Experience with testing frameworks like Jest, Mocha, or Jasmine
  • UX/UI design principles and best practices
  • Performance optimization techniques for frontend code
  • Accessibility standards and practices (WCAG)
  • Cross-browser testing and debugging skills
  • Continuous integration/continuous deployment (CI/CD) workflows

Job Responsibility :

  • Develop and maintain responsive web applications with a focus on frontend technologies.
  • Collaborate with designers and backend developers to implement user-friendly interfaces and ensure seamless integration with backend systems.
  • Write clean, efficient, and well-documented HTML, CSS, and JavaScript code.
  • Translate design mockups and wireframes into high-quality, interactive user interfaces.
  • Optimize frontend code for performance and scalability.
  • Conduct thorough testing and debugging to ensure cross-browser and cross-device compatibility.
  • Stay updated on industry trends and best practices to continuously improve frontend development processes.
  • Work closely with product managers and stakeholders to understand project requirements and deliver user-centric solutions.
  • Troubleshoot and resolve frontend-related issues in a timely manner.
  • Participate in code reviews to maintain code quality and adherence to coding standards.
  • Collaborate with the UX/UI team to enhance user experiences and interface designs.
  • Contribute to the architectural design and implementation of frontend systems.
  • Mentor junior developers and provide technical guidance when needed.
  • Implement accessibility standards and ensure web applications are usable by all users.
  • Assist in the deployment and maintenance of web applications in production environments.
